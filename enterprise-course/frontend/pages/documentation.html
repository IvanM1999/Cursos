Introdução

Enterprise Course é uma plataforma completa para gerenciamento de cursos e conteúdo, construída com Node.js, Express e suporte para MongoDB e PostgreSQL.

Características

  • Autenticação JWT e sessões
  • CRUD de blogs com editor Markdown
  • Dicionário de serviços/APIs
  • Sistema de usuários
  • Escalável e modular

Arquitetura

A aplicação segue uma arquitetura em camadas:

  • Core: Núcleo com gerencimento de conexões
  • Brain: Lógica de negócios
  • Routes: Rotas da API
  • Middleware: Middlewares de autenticação e validação
  • Frontend: Interface com HTML e JavaScript

Setup Inicial

# Instalar dependências
npm install

# Copiar variáveis de ambiente
cp .env.example .env

# Editar .env com suas configurações

# Iniciar em desenvolvimento
npm run dev

# Iniciar em produção
npm start

Autenticação

JWT

Para requisições API, inclua o token JWT no header:

Authorization: Bearer seu_token_aqui

Sessão

As páginas HTML usam sessão de servidor para autenticação.

APIs Disponíveis

Autenticação

  • POST /api/auth/signup - Criar conta
  • POST /api/auth/login - Fazer login
  • POST /api/auth/logout - Fazer logout

Blogs

  • GET /api/blogs - Listar blogs
  • POST /api/blogs - Criar blog
  • PUT /api/blogs/:id - Atualizar blog
  • DELETE /api/blogs/:id - Deletar blog

Banco de Dados

O projeto suporta MongoDB e PostgreSQL. Configure em .env:

MongoDB

MONGODB_URI=mongodb://localhost:27017/enterprise_course

PostgreSQL

DB_HOST=localhost
DB_PORT=5432
DB_NAME=enterprise_course
DB_USER=postgres
DB_PASSWORD=sua_senha

Segurança

  • Senhas hasheadas com bcrypt
  • CORS configurado
  • Helmet para headers de segurança
  • Validação de entrada
  • JWT com expiração

Deployment

Heroku

git push heroku main

Docker

docker build -t enterprise-course .
docker run -p 3000:3000 enterprise-course
